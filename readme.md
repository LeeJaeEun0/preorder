# 예약 구매하기 프로젝트

## 1. 예약 구매

- 사용자간 커뮤니티를 할 수 있는 소셜미디어 서비스를 제공
    - 게시글 작성, 댓글 작성, 좋아요, 팔로우, 뉴스피드 가능
- 상품을 구매할 수 있는 서비스 제공
    - 일반 상품과 정해진 시간에 상품을 구매할 수 있는 예약 상품 구매 가능
